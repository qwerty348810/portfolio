# dmitryyakovlev.by

Личный сайт-портфолио. Веб-разработка и SEO для производителей климатического и торгового оборудования.

🔗 **Живая версия:** [dmitryyakovlev.by](https://dmitryyakovlev.by)

## Стек

- Vanilla HTML/CSS/JS — без зависимостей и сборки
- Google Fonts: Inter, JetBrains Mono
- IntersectionObserver для активной секции в навигации
- CSS-переменные для палитры и spotlight-эффекта

## Локальный запуск

Откройте `index.html` в браузере. Сборка не требуется.

Для удобства разработки можно поднять любой статический сервер:

```bash
npx serve .
# или
python3 -m http.server 8000
```
